Para la ejecucion del programa la rutina recibe los siguientes parametros:

ImgN : Cadena de Caracteres en el cual se almacena el nombre de la imagen incluyendo su extension de archivo
Mit  : Entero, en el cual se almacena el maximo de iteraciones
Lmd  : Flotante, en el cual se almacena el parametro de regularizacion Lamnda
Id   : Entero, el cual se almacena para identificar la imagen salida

de esta manera, los programas Suavizado.cpp y SecSuavizado.cpp se ejecutan como sigue:

./Suavizado <ImgN> <Mit> <Lmd> <Id>

./SecSuavizado <ImgN> <Mit> <Lmd> <Id>
